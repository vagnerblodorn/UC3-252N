3.explique a diferença de um Banco de Dados, um sistema gerenciador de banco de dados e um sistemas de banco de dados?
R=Um banco de dados é a coleção de dados organizados; um Sistema Gerenciador de Banco de Dados (SGBD) é o software que manipula e gerencia esses dados; e um Sistema de Banco de Dados é a combinação do banco de dados e do SGBD, formando um sistema completo. A principal diferença é que o SGBD é a ferramenta (o "software") e o banco de dados é o "conteúdo" (a coleção de dados). 
4. quais são as principais características de um banco de dados ?
R=As principais características de um banco de dados são organização, eficiência, escalabilidade e segurança. Eles armazenam dados de forma estruturada para fácil gerenciamento, permitem recuperação e atualização rápidas, podem lidar com volumes crescentes de dados e protegem informações confidenciais com controles de acesso robustos. 
5 quais são os diferentes niveis de abstacao de banco de dados?
R= Os três níveis de abstração de banco de dados são o interno (ou físico), o conceitual (ou lógico) e o externo (ou de visualização). 
6. Liste as vantagens de usar banco de dados?
R=As vantagens de usar bancos de dados incluem organização, agilidade, segurança, acesso facilitado, redução de redundância, confiabilidade, escalabilidade, suporte à tomada de decisões, colaboração e conformidade com leis.
